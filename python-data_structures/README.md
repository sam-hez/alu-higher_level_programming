# Python - Data Structures: Lists, Tuples

This project covers the basic data structures in Python, focusing on Lists and Tuples.

## Learning Objectives
- Understanding Python lists and tuples.
- Similarities and differences between strings and lists.
- Common list methods.
- Using lists as stacks and queues.
- List comprehensions.
- Sequence unpacking and tuple packing.
- Using the `del` statement.

## Requirements
- All files are interpreted on Ubuntu 20.04 LTS using `python3` (3.8.5).
- Code follows `pycodestyle` (version 2.7.*).
- All files are executable.
