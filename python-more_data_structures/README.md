# Python - More Data Structures: Set, Dictionary

This project covers more advanced data structures in Python, including sets, dictionaries, and functional programming concepts like `map`, `filter`, and `lambda`.

## Learning Objectives
- Understanding Python sets and dictionaries.
- Common methods for sets and dictionaries.
- When to use sets vs. lists vs. dictionaries.
- Iterating through sets and dictionaries.
- Lambda functions.
- `map`, `filter`, and `reduce` functions.

## Requirements
- All files are interpreted on Ubuntu 20.04 LTS using `python3` (3.8.5).
- Code follows `pycodestyle` (version 2.7.*).
- All files are executable.
